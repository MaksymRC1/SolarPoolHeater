# Solar Pool Heater Controller

Arduino контроллер для управления нагревом уличного бассейна от солнечного коллектора.

## Компоненты
- Arduino Uno
- LCD 16x2
- 2x DS18B20 (вход и выход коллектора)
- DHT22 (температура и влажность воздуха)
- Реле (управление насосом)
- 2 кнопки
- Зуммер

## Режимы работы
- **AUTO** — автоматическое включение насоса по разнице температур (ΔT)
- **MANUAL** — поддержание заданной температуры бассейна

## Управление
- Короткое нажатие обеих кнопок — переключение экрана
- Короткое нажатие «+» / «-» — изменение параметра
- Долгое нажатие (5 сек) обеих кнопок — переключение режима

## Сборка
Проект создан в PlatformIO. Для сборки выполните:
\`\`\`bash
pio run
\`\`\`

## Загрузка на Arduino
\`\`\`bash
pio run --target upload
\`\`\`
